SchemaCrawler is a platform (operating system and database) independent command-
line tool to output your database schema and data in a readable form. The output 
is designed to be diff-ed with previous versions of your database schema. 
SchemaCrawler has powerful grep  functionality that allows you to search for 
table and column names using regular expressions. SchemaCrawler can also 
generate schema diagrams.
