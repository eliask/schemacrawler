--- Command ---

-c=freemarker (short for -command=freemarker)
  Processes a FreeMarker template
  
--- Output Options ---

-outputformat=<outputformat>
	Where <outputformat> is the file path or class path to the FreeMarker 
	template

-o=<outputfile> (short for -outputfile=<outputfile>)
	Optional 
	<outputfile> is the path to the output file
