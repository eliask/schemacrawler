--- Configuration Options ---

-g <config-file> (short for -configfile <config-file>)
  Optional, reads SchemaCrawler configuration properties from <config-file> 
  instead of the default schemacrawler.config.properties file.

-p <config-override-file> (short for -configoverridefile <config-override-file>)
  Optional, reads SchemaCrawler configuration properties from 
  <config-override-file> and overrides the properties from the 
  configuration file.
